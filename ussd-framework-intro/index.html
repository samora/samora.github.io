Intro to SMSGH USSD Framework

Build and maintain USSD apps with ease and sanity.

Presented by Samora Dake / @papasamo247

Purpose

  • Speed up USSD client development
  • Sane maintenance of USSD clients
  • Minimize spaghetti code
  • Provide USSD clients with a rich feature set out of the box.

Features

  • Session management
  • Menu/screen navigation
  • Resume timed out USSD sessions at last screen
  • Selection based inputs
  • Fully asycnchronous
  • VS 2013 project template
